# Framer Animated Portfolio Template

A modern, animated portfolio template for the Funnel Builder Website Builder. Features:

- Animated hero section (fade-in)
- Scroll-triggered section animations (slide-up, fade-in)
- Interactive project cards (hover/expand)
- Modular sections: Hero, About, Projects, Contact
- Designed for easy import and live preview

## Files
- `template.json`: Template structure, animation metadata, and content
- `preview.png`: Visual preview (add your image)
- `hero-bg.jpg`, `project1.jpg`, `project2.jpg`, `project3.jpg`: Example images (replace with your own or connect to media library)

## Usage
- Import this template into the Website Builder
- Animations and interactivity are defined in the JSON and can be mapped to GrapesJS/Craft.js blocks
- Extend or customize sections as needed
