# External Templates

This folder will contain open-source, world-class website templates (Framer, Webflow, Tailwind UI, etc.) for import into the builder. Each template will include:

- Multiple sections and layouts
- Animations (e.g., fade-in, parallax, scroll effects)
- Interactive elements (tabs, sliders, accordions, video backgrounds)
- Metadata for animation and interactivity

## How to Add a Template
- Place each template in its own subfolder (e.g., `framer-portfolio`, `webflow-agency`, `tailwind-saas`)
- Include a `template.json` with structure, animation, and interactivity metadata
- Include any assets (images, CSS, JS) needed for the template

## Example Structure
```
external-templates/
  framer-portfolio/
    template.json
    preview.png
    ...
  webflow-agency/
    template.json
    preview.png
    ...
```
